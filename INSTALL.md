# TopAnimation - Все версии модов

## 📝 Описание
**TopAnimation** - профессиональный мод для Minecraft с реалистичными анимациями:
- ⚔️ Реалистичные анимации мечей с физикой взмаха
- 🚶 Плавные анимации ходьбы и бега игроков
- 👀 POV камера без отображения тела
- 💧 Реалистичная физика воды
- 🎨 Полная интеграция с OptiFine и FreshAnimations

## 🚀 Установка

### Forge
1. Убедитесь, что установлен Forge для вашей версии Minecraft
2. Скопируйте JAR файл в папку `%appdata%/.minecraft/mods` (Windows) или `~/.minecraft/mods` (Mac/Linux)
3. Запустите Minecraft

### Fabric
1. Установите Fabric Loader для вашей версии
2. Скопируйте JAR файл в папку `mods`
3. Запустите Minecraft

## 📋 Требования
- **Minecraft**: 1.16.5 - 1.26.1
- **Java**:
  - 1.16.5-1.19: Java 11+
  - 1.20-1.21: Java 16+
  - 1.26.1: Java 17+
- **Forge/Fabric**: Последняя версия для вашей версии Minecraft
- **OptiFine** (рекомендуется)
- **FreshAnimations** (опционально)

## ⚡ Особенности

### Анимации мечей
- Реалистичный взмах вместо простого вверх-вниз
- Трёхфазная анимация (разгон, максимальная скорость, торможение)
- Физика инерции
- Колющие удары
- Вращающиеся атаки

### Анимации игроков
- Плавная ходьба с качанием рук
- Быстрый бег с прыгающим движением
- Спокойное стояние с лёгким покачиванием
- Синхронизированные движения

### Вода
- Реалистичные волны
- Течение жидкостей
- Улучшенные визуальные эффекты
- Интеграция с шейдерами

### POV Камера
- Автоматическое скрытие тела в POV
- Видны только руки с оружием
- Плавные переходы
- Реалистичный угол зрения

## 💻 Как загрузить на Modrinth

1. Перейдите на https://modrinth.com
2. Нажмите "Create Project"
3. Выберите "Mod"
4. Заполните информацию:
   - **Название**: TopAnimation
   - **Описание**: Realistic Animations for Minecraft
   - **Версия**: 1.0.0
5. Загрузите JAR файлы для нужных версий
6. Выберите категории: Animation, Client-side
7. Нажмите "Create"
