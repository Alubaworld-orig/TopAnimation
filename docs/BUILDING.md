# Сборка TopAnimation

## Требования
- JDK 17+
- Gradle 7.0+

## Для Forge
```bash
cd 1.26.1-forge
./gradlew build
```

## Для Fabric
```bash
cd 1.26.1-fabric
./gradlew build
```

## Выходные файлы
- `build/libs/` - JAR файлы модов

## Все версии
Повторите процесс для каждой папки версии (1.16.5, 1.17, 1.18, 1.19, 1.20, 1.21, 1.26.1)
