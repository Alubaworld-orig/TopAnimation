# TopAnimation Resource Pack

## Совместимость
- OptiFine
- FreshAnimations
- Minecraft 1.16.5 - 1.26.1

## Установка
1. Загрузите ресурспак
2. Поместите в папку resourcepacks
3. Активируйте в Settings > Resource Packs

## Содержимое
- Анимации мечей
- Анимации рук и ног
- Текстуры воды с реалистичными волнами
- Конфиги для OptiFine

## Требования модов
- TopAnimation mod (основной мод)
