# TopAnimation - Realistic Minecraft Mod

## Описание
Mod TopAnimation добавляет реалистичные анимации в Minecraft:
- ⚔️ Реалистичные анимации мечей с физикой взмаха
- 🚶 Плавные анимации игроков
- 👁️ POV камера без отображения тела
- 💧 Реалистичная физика воды
- 🎨 Интеграция с FreshAnimations и OptiFine

## Поддерживаемые версии
- Minecraft 1.16.5 - 1.26.1
- Forge & Fabric

## Требования
- Forge/Fabric Loader
- Optifine (рекомендуется)
- FreshAnimations (опционально)

## Установка
1. Загрузите JAR для вашей версии
2. Поместите в папку mods
3. Запустите игру
4. Установите ресурспак (если нужен)

## Особенности
### Анимации мечей
- Реалистичный взмах вместо простого вверх-вниз
- Физика инерции и затухания
- Разные анимации для холодного оружия

### POV Камера
- Без отображения тела игрока
- Реалистичные руки с оружием
- Плавные переходы анимаций

### Вода
- Реалистичная физика жидкостей
- Волны и течения
- Улучшенные визуальные эффекты
