# TopAnimation - Реалистичные Анимации для Minecraft

## 📝 Описание
**TopAnimation** - профессиональный мод для Minecraft, добавляющий реалистичные анимации:
- ⚔️ Реалистичные анимации мечей с физикой взмаха
- 🚶 Плавные анимации ходьбы и бега игроков
- 👀 POV камера без отображения тела
- 💧 Реалистичная физика и анимация воды
- 🎨 Полная интеграция с OptiFine и FreshAnimations

## 🚀 Установка

### Forge
1. Загрузите нужный JAR файл для вашей в��рсии
2. Поместите в папку `%appdata%/.minecraft/mods` (Windows) или `~/.minecraft/mods` (Linux/Mac)
3. Запустите Minecraft с установленным Forge
4. Готово!

### Fabric
1. Убедитесь, что установлен Fabric Loader
2. Загрузите нужный JAR файл для вашей версии
3. Поместите в папку `mods`
4. Запустите игру
5. Готово!

## 📋 Требования
- **Minecraft**: 1.16.5 - 1.26.1
- **Forge/Fabric**: Последняя версия для вашей версии Minecraft
- **Java**: 
  - 1.16.5-1.19: Java 11+
  - 1.20: Java 16+
  - 1.21+: Java 17+
- **Optifine** (рекомендуется)
- **FreshAnimations** (опционально)

## ⚙️ Характеристики

### Анимации мечей
- Реалистичный взмах вместо простого вверх-вниз
- Трёхфазная анимация (разгон, максимальная скорость, торможение)
- Физика инерции
- Колющие удары
- Вращающие атаки

### Анимации игроков
- Плавная ходьба с качанием рук
- Быстрый бег с прыгающим движением
- Спокойное стояние с лёгким покачиванием
- Синхронизированные движения

### Вода
- Реалистичные волны
- Течение жидкостей
- Улучшенные визуальные эффекты
- Интеграция с шейдерами

### POV Камера
- Автоматическое скрытие тела в POV
- Видны только руки с оружием
- Плавные переходы
- Реалистичный угол зрения

## 🎮 Как использовать

1. **Выберите версию**: Определите, на какой версии Minecraft вы играете
2. **Выберите лоадер**: Используете ли вы Forge или Fabric
3. **Загрузите JAR**: Возьмите нужный файл из архива
4. **Поместите в mods**: Скопируйте в папку `mods` вашего Minecraft
5. **Запустите игру**: Мод автоматически загрузится
